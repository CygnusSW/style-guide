Cygnus Stylesheet

The Cygnus Stylesheet is a living document capturing the basing tools used to create applications in cygnus software.

Variables

Override these variables for your project:

  • $base-font-size (default: 18pt)
  • $num-of-grid-cols (default: 12)
  • $breakpoints (A map of min and max breakpoints.)
    • xs: (none, none)
    • s: (300px, null)
    • m: (815px, null)
    • l: (1028px, null)
    • xl: (1366px, null)
  • $header-font-family (default: "Fahkwang", sans-serif)
  • $body-font-family (default: "PT Sans", sans-serif)

Instructions

  1. npm install --save cygnus-stylesheet
  2. If extending the SCSS, configure your projects variables
  3. <link rel="stylesheet" src="node_modules/cygnus-stylesheet/scss/main.css" > OR @import "node-modules/cygnus-stylesheet/scss/main.scss"
